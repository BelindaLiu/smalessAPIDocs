Panels

Basic panels

By default, all the.panel does is apply some basic border and padding to contain some content.

See the PenBasic panels-0.1.0 by Sam (@xiang) onCodePen.

Panel with heading

Easily add a heading container to your panel with.panel-header . You may also include any< h1 >-< h6 > with a.panel-title class to add a pre-styled heading.

See the PenPanel with heading-0.1.0 by Sam (@xiang) onCodePen.

Panel with footer

Wrap buttons or secondary text in.panel-footer . Note that panel footers do not inherit colors and borders when using contextual variations as they are not meant to be in the foreground.

See the PenPanel with footer-0.1.0 by Sam (@xiang) onCodePen.

With tables

Add any non-bordered.table within a panel for a seamless design. If there is a.panel-body, we add an extra border to the top of the table for separation.

See the PenWith tables-0.1.0 by Sam (@xiang) onCodePen.

Tables

Base Tables

For basic styling-light padding and only horizontal dividers-add the base class.table to any< table >. It may seem super redundant,but given hte widespread use of tables for other plugins like calendars and date pickers,we've opted to isolate our custom table styles.

See the PenBase Tables-0.1.0 by Sam (@xiang) onCodePen.

Tables details

Use.detail-title to displays detailed information in a table.

See the PenTables details-0.1.0 by Sam (@xiang) onCodePen.
